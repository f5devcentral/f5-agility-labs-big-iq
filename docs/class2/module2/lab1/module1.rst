Module 1: Analytics in BIG-IQ
=============================

BIG-IQ 6.0 preserve all the Analytics from the previous release 5.4, under the **Monitoring** Tab: Device Health, DNS, Access, WAF.

With the new release, BIG-IQ bring the Analytics per Application under the **Application** tab.
